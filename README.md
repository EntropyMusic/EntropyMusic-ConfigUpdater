# Entropy Music Firmware Configurator

A standalone desktop app for configuring [Entropy Music](https://entropymusic.cc)
hardware over USB MIDI. It is a thin Electron wrapper around the firmware page of
the Entropy Music website, packaged so it runs offline as a native app on macOS,
Windows, and Linux.

Pick your device, edit each control's MIDI channel, CC, pitch-bend mode, name, and
LED colors per bank, then upload the configuration straight to the hardware via MIDI
SysEx.

## Why a desktop app?

Web MIDI SysEx access is restricted in browsers. Shipping the configurator as an
Electron app under a custom secure `app://` scheme grants reliable SysEx access and
lets the tool work without an internet connection (aside from one optional icon
font, see below).

## How it works

The UI is **not** maintained here. The `web/` git submodule is the Entropy Music
website, and it is the source of truth for all JavaScript and CSS. `npm run sync`
([scripts/sync-firmware.mjs](scripts/sync-firmware.mjs)) vendors only the firmware
page and its runtime dependencies into `renderer/`. That sync runs automatically
before `npm start` and every packaging command.

Files authored directly in this repo (never overwritten by sync):

- [main.js](main.js) — Electron main process: window, `app://` scheme, native menu.
- `renderer/index.html` — the firmware editor page.
- `renderer/launcher.html` — the "Select Your Device" home page.

Everything under `renderer/{js,css,components,assets}` is generated by the sync and
is gitignored. To change app behavior or styling, edit the `web/` submodule and
re-run `npm run sync`.

## Getting started

```bash
git submodule update --init   # fetch the web/ submodule
npm install
npm start                     # runs the sync automatically, then launches
```

## Building installers

```bash
npm run dist          # current platform
npm run dist:mac      # .dmg
npm run dist:win      # NSIS installer
npm run dist:linux    # AppImage
```

Output lands in `dist/`.

## Notes

- The editor loads Font Awesome from a CDN for a couple of icons. Swap it for a
  local copy in `renderer/index.html` if you need a fully offline build.
- App settings (accent color, reduce motion, auto-sync, confirm-before-upload,
  default view) persist in `localStorage` and are reachable from the gear button on
  either page or the native **Settings…** menu item (`Cmd`/`Ctrl` + `,`).
